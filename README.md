# Verejný archív SuŠi

Úložisko materiálov z predošlých ročníkov SuŠi. Obsahuje zdrojové kódy a PDF zadaní a vzorákov, staré výsledky a ďalšie materiály (a scripty, ktorými boli získané z pôvodných stránok a repozitárov).

Aktuálne súťaže, ktoré organizuje SuŠi, nájdete na [https://susi.sk](https://susi.sk). 

## Organizátori SuŠi:

Ľudia, ktorí prispeli tvorbou šifier, recenzovaním, vymýšlaním nápadov, testovaním, pomocou s technickým zabezpečením, vedúcovaním sústredení...
